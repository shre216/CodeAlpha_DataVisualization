# 📊 Data Visualization Project
## 🚀 Global Superstore Analysis using Python

---

## 🌟 Overview
This project demonstrates **data visualization techniques** using Python on the **Global Superstore dataset**.  
It converts raw data into meaningful insights using different types of charts.

---

## 🧰 Tech Stack
- Python 🐍
- Pandas
- Matplotlib
- Seaborn

---

## 📁 Dataset
- File: `Global Superstore.csv`
- Includes:
  - Sales
  - Category
  - Region
  - Product Name
  - Order Date
